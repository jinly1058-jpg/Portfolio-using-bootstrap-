Coffee Shop Website

Modern Responsive Coffee Shop Website

Project Overview

The Coffee Shop Website is a responsive web application designed for coffee shops to showcase their menu, services, special offers, and contact information. Customers can explore products and learn more about the coffee shop online.

Features

  • Responsive Design
  • Home, About & Menu Pages
  • Special Offers Section
  • Gallery Section
  • Customer Reviews
  • Contact Form

Technologies Used

HTML CSS Bootstrap 5 JavaScript
