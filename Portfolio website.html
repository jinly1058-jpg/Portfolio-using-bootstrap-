Portfolio Website

Personal Portfolio Website Using Bootstrap 5

Project Overview

This Portfolio Website was developed to showcase my skills, projects, certificates, education, and contact information in a professional way. The website is fully responsive and built using Bootstrap 5.

Features

  • Responsive Design
  • Animated Profile Section
  • About Me Section
  • Skills with Progress Bars
  • Projects Showcase
  • Certificates Section
  • Resume Download Option
  • Contact Form
  • Social Media Integration

Technologies Used

HTML CSS Bootstrap 5 JavaScript

Project Goal

The main goal of this project is to create a personal branding platform that helps employers and clients understand my technical skills and project experience.
