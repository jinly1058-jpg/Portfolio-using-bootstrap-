Student Management System

Modern Responsive Student Management Website

Project Overview

The Student Management System is a responsive web application designed to manage student information efficiently. It helps schools and colleges maintain student records, attendance, courses, and results in a simple and organized way.

Features

  • Student Registration
  • Student Details Management
  • Attendance Tracking
  • Course Management
  • Marks and Results Management
  • Teacher Information
  • Contact Form

Technologies Used

HTML CSS Bootstrap 5 JavaScript
